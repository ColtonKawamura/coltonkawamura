# Expansions
## Taylor Series Expansions
Expanding $f(x)$ about a point $a$
$$\begin{align*}
f(x) = f(a) + f'(a)(x - a) + \frac{f''(a)}{2!}(x - a)^2 + \frac{f'''(a)}{3!}(x - a)^3 + \cdots + \frac{f^{(n)}(a)}{n!}(x - a)^n + \cdots
\end{align*}$$
